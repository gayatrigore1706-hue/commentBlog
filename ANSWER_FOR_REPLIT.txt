## CommentBlog - Django Blog Platform

**CommentBlog** is a fully-functional Django web application for blogging.

### What It Does:
- Users can create, edit, and delete blog posts
- Posts are organized by categories
- Users can add comments to any post
- User authentication with login/registration
- Personal dashboard for managing own posts
- Admin panel for site moderators
- Support for uploading featured images
- Mobile-responsive design

### Technology:
- **Language:** Python 3.12
- **Framework:** Django 5.2.8
- **Database:** SQLite (auto-created, persistent)
- **Frontend:** HTML5, CSS3, JavaScript
- **Libraries:** Pillow (for images), WhiteNoise (for static files)
