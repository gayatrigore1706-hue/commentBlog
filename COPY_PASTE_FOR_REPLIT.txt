CommentBlog is a full-featured Django blog platform built with Python and Django 5.2.8.

**What it does:**
- Users can create, edit, and delete blog posts
- Posts organized by categories
- Comments system on posts
- User authentication and registration
- Personal dashboard for managing posts
- Admin panel for moderation
- Image uploads for post featured images
- Responsive web design

**Technology:** Python + Django + SQLite + HTML5/CSS3

**Setup in Replit:**
1. Upload the ZIP or import from GitHub
2. In Shell, run:
   pip install -r requirements.txt
   python CommentBlog/manage.py migrate
   python CommentBlog/manage.py runserver 0.0.0.0:8000

3. Your public URL will appear - that's your global blog link!

**Files included:**
- requirements.txt with all dependencies
- .replit configuration
- Complete Django application
- Documentation and setup guides

**Sample users available after running populate_data:**
- admin / admin123 (administrator)
- user1, user2, user3 / password123

Ready to deploy! Just follow the setup steps.
