## Setup Instructions for Replit

### Step 1: Navigate to Project
```bash
cd CommentBlog
```

### Step 2: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 3: Initialize Database
```bash
python manage.py migrate
```

### Step 4: Load Sample Data (Optional)
```bash
python manage.py populate_data
```

Creates sample users and blog posts:
- Admin user: `admin` / `admin123`
- Test users: `user1`, `user2`, `user3` / `password123`

### Step 5: Run the Server
```bash
python manage.py runserver 0.0.0.0:8000
```

If you get any errors:

1. **Missing modules:** `pip install -r requirements.txt`
2. **Database errors:** `python manage.py migrate`
3. **Port in use:** Port 8000 will be used automatically
